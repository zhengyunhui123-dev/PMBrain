# PM 项目状态查询

## 功能
汇总项目健康度、里程碑进度、阻塞项、风险概览。

## 执行步骤
1. 读取根目录下的 schema.yaml，获取 PM 页面类型定义
2. 使用 search 工具搜索所有 type: project 的页面，获取项目列表
3. 对每个项目，搜索其关联的 milestone、task、risk 页面
4. 汇总以下信息：
   - 项目整体状态（正常/有风险/严重）
   - 各里程碑完成进度
   - 超期或阻塞的任务
   - 高概率高影响的风险项
5. 输出结构化摘要，按项目分组展示

## 输出格式
按项目分组，每个项目包含：
- 状态标识（🟢正常 / 🟡有风险 / 🔴严重）
- 里程碑进度条
- 阻塞任务列表
- 重点关注的风险
